# SMART MOUSE - Virtual Mouse with Voice Assistant

## 📌 Project Overview
SMART MOUSE is an AI-based virtual mouse system that allows users to control mouse functions using **hand gestures** and execute system or web actions using a **voice assistant**.  
This project combines **Computer Vision**, **Gesture Recognition**, and **Voice Commands** to provide a touchless human-computer interaction experience.

---

## 🚀 Features
- 🖐️ Hand gesture-based virtual mouse control
- 🎯 Cursor movement using finger tracking
- 👆 Left click and right click using gestures
- 🔊 Voice assistant for system and web commands
- 🌐 Open websites like YouTube, Gmail, Spotify, etc.
- ⚙️ Open system tools like Settings and Task Manager
- 🖥️ Simple web interface to launch functionalities
