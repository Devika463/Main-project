# 🖐️ Virtual Mouse Using Hand Gesture Recognition

🔹 Touch-free virtual mouse controlled using real-time hand gestures  
🔹 Uses a webcam and computer vision for intuitive interaction  
🔹 Eliminates the need for physical mouse hardware  

## ⚙️ Key Functionalities
✔ Real-time hand detection and tracking  
✔ Smooth cursor movement using finger gestures  
✔ Left click using thumb and index finger  
✔ Right click using index and middle finger  
✔ Scroll control using multi-finger gestures  
✔ Pause and resume cursor movement  

## ✨ Features
★ Contactless and user-friendly interface  
★ No additional hardware required  
★ Real-time performance with minimal latency  
★ Cost-effective and easy to implement  
★ Suitable for academic and practical use
